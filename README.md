# PROG6212POE

This application is a claims system. Whereby lecturers can submit overtime claims for approval from management of different roles

Roles
Lectureres - submits claims
Programme coordinator - Forwards claims to academic manager
Academic manager -Approves forwarded claims
HR - Can create users as well as export the data of all approved claims to a csv 

For this part of the poe, login features using identity was added. This means than users can only see the relevant information attached to them. Also added was the entire HR role which can (as previously stated) add employees as well as export claim data
